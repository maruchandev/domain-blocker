# Domain Blocker

指定したドメインへのアクセスをブロック・許可できる Chrome 拡張機能です。
シンプルでプライバシー重視。余計な通信は一切行いません。

## 機能

- **ブロックリスト** — 指定したドメインへのアクセスをブロック
- **許可リスト** — ブロック対象でも常にアクセスを許可するドメインを指定
- **個別 ON/OFF** — ブロックリストの各ドメインを一時的に無効化可能
- **リダイレクト** — ブロック時に任意のURLへ転送
- **PIN 保護** — 設定画面へのアクセスを 4 桁の PIN で保護（初期値: 0000）
- **ポップアップ連携** — 現在のサイトの状態を表示し、設定画面へクイックアクセス

## 使い方

### インストール（開発者モード）

1. [リポジトリ](https://github.com/maruchandev/domain-blocker) からダウンロードまたはクローン
2. Chrome で `chrome://extensions` を開く
3. 右上の「デベロッパーモード」を ON
4. 「パッケージ化されていない拡張機能を読み込む」からフォルダを選択

### 基本的な操作

| 操作 | 方法 |
|---|---|
| 拡張機能の ON/OFF | ツールバーのアイコン → トグルスイッチ |
| 現在のサイトを許可/ブロック | ポップアップのボタン → 設定画面で追加 |
| リストの管理 | アイコン右クリック → オプション、またはポップアップの「Settings」 |
| PINの変更 | 設定画面 → Change PIN |

## ファイル構成

```
domain-blocker/
├── manifest.json      # 拡張機能の設定
├── background.js      # バックグラウンド Service Worker
├── popup.html/.js     # ポップアップ UI
├── options.html/.js   # 設定画面 UI
└── icon*.png          # アイコン
```

## プライバシー

- 外部サーバーとの通信は一切行いません
- データはすべて Chrome のローカルストレージ (`storage.sync`) に保存されます
- ブロック処理は Chrome の `declarativeNetRequest` API によりブラウザレベルで実行されます
